## 🛠 Hướng dẫn Chạy Code (Localhost)

Để xem website hoạt động trên máy tính, bạn nên sử dụng **Visual Studio Code** để có trải nghiệm tốt nhất:

1.  **Mở thư mục:** Mở thư mục chứa mã nguồn bằng phần mềm VS Code.
2.  **Cài đặt Live Server:** Vào mục Extensions (phím tắt `Ctrl+Shift+X`), tìm và cài đặt plugin tên là **Live Server**.
3.  **Khởi chạy:** 
    *   Mở file `index.html`.
    *   Nhấn chuột phải chọn **Open with Live Server** hoặc nhấn nút **Go Live** ở góc dưới cùng bên phải màn hình.
4.  **Kết quả:** Trình duyệt sẽ tự động mở website tại địa chỉ `http://127.0.0.1:5500`.

## 🚀 Hướng dẫn Deploy (Đưa lên mạng miễn phí)

Bạn có thể đưa website lên internet chỉ trong "vài nốt nhạc" qua các nền tảng sau:

*   **Vercel / Netlify:** Truy cập trang web của họ, đăng nhập và kéo thả toàn bộ thư mục code vào vùng upload. Website của bạn sẽ có link truy cập công khai ngay lập tức.
*   **GitHub Pages:** Tải code lên một Repository trên GitHub, sau đó vào **Settings > Pages** để kích hoạt tính năng lưu trữ web.

## 🧪 Cách Test Code

*   **Responsive Test:** Nhấn `F12` (hoặc `Ctrl+Shift+I`), chọn biểu tượng điện thoại/máy tính bảng để kiểm tra xem giao diện có bị vỡ trên các thiết bị khác nhau không.
*   **Console Check:** Kiểm tra tab **Console** để đảm bảo không có lỗi script nào gây ảnh hưởng đến tính năng của website.
